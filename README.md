# SIT305-3.1C

## Project Description
This Android application is a multiple choice quiz app developed using Java in Android Studio. The app allows users to enter their name, attempt a quiz, track progress, and view their final score.

---

## Features Implemented

### Subtask 1: Answer Selection & Feedback
- Correct answer turns **green**
- Incorrect answer turns **red**
- User cannot change answer after submission

---

### Subtask 2: Real Time Progress Tracking
- Progress bar updates dynamically
- Displays **question number (e.g., 2 of 5)**

---

### Subtask 3: Final Score & Session Persistence
- Displays final score (e.g., 4/5)
- Shows **user name**
- “Take New Quiz” button returns to main screen
- “Finish” button closes the app

---

## Technologies Used
- Java
- Android Studio
- XML (UI Design)
